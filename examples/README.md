# Examples

This directory contains example usage of Capiscio A2A Security.

## Running Examples

```bash
# Install the package in development mode
pip install -e .

# Run an example
python examples/01_minimal_integration.py
```

## Available Examples

- `01_minimal_integration.py` - Minimal one-liner integration
- `02_explicit_config.py` - Explicit configuration pattern
- `03_decorator_pattern.py` - Decorator-based integration
- `04_adk_agent.py` - Google ADK agent integration
- `05_with_x402.py` - Combined with x402 payments
- `06_environment_config.py` - Environment-based configuration

*(Examples will be added as implementation progresses)*
